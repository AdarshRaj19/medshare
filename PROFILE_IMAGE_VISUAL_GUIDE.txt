Step 1: USER LOGS IN
    ↓
    See navbar with default user icon
    ┌─────────────────────────────────────┐

    └─────────────────────────────────────┘

Step 2: USER GOES TO PROFILE
    ↓
    Click navbar avatar OR settings icon
    ↓
    See profile page with upload form
    ┌───────────────────────────────────┐
    │ [👤 Default] John Doe             │
    │ [📷 Choose File] [💾 Save]        │
    └───────────────────────────────────┘

Step 3: USER UPLOADS IMAGE
    ↓
    Select JPG/PNG file
    ↓
    Click "Save Changes"
    ↓
    Django:
    - Validates image
    - Saves to media/profiles/{id}/image.jpg
    - Updates database
    ↓
    Success message shows

Step 4: PAGE RELOADS
    ↓
    Profile page displays image:
    ┌─────────────────────────────┐
    │ 🖼️ (120x120)  John Doe    │
    │ @johndoe • Donor           │
    │ City Hospital              │
    └─────────────────────────────┘

Step 5: USER SEES NAVBAR
    ↓
    Navbar now shows avatar:
    ┌──────────────────────────────────┐
    │ [Logo] ... [🔔] [🌙] [👤 Photo] │
    └──────────────────────────────────┘

Step 6: USER CLICKS AVATAR
    ↓
    Goes back to profile page
    ✅ Seamless user experience!
